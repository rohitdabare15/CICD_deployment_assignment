Part 1: Deploy Flask and Express on a Single EC2 Instance
# Flask + Express CI/CD Deployment Project

This project demonstrates:
- Deploying Flask backend on port 5000
- Deploying Express frontend on port 3000
- Automating deployment using Jenkins CI/CD

## Architecture

GitHub → Jenkins (Windows) → AWS EC2 → Flask + Express

## Backend
Runs on port 5000

## Frontend
Runs on port 3000
